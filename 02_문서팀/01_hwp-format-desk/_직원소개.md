# 01_hwp-format-desk — 한글서식 수정원

② 문서팀 소속. 한글(HWP/HWPX) 문서의 서식을 가변·대화형으로 손보는 직원.

## 하는 일
- **가변 서식 수정** — "이 표만 글꼴 바꿔", "A 서식을 B 문서에 적용", "캡션 다시 매겨" 등
  exe로는 못 하는 즉석 서식 작업. → 직원이 `hwp-mcp`로 직접 처리.
- **다단계 흐름의 변환 단계** — "검토 → pdf → 첨부" 같은 파이프라인의 변환 담당.

## 하지 않는 일 (exe가 더 빠름 — 경로만 안내)
- **단독 HWP→PDF 변환** → `HwpPdfTool.exe` 더블클릭이 최선.
  변환 병목은 한글 렌더링 시간이라 직원이 해도 동일하고 오버헤드만 늘어남.
- **고정 서식 통일** (표/그림 캡션 번호) → `HwpFormatter.exe`.
  `서식테스트.xlsm` 규칙 기반 결정론적 변환이라 판단할 게 없음.

## 도구
- `hwp-mcp` (한글 즉석 제어), `pyhwpx` (색·음영 세밀 조작)
- 기존 exe 2종을 직원의 "도구"로 안내: `HwpPdfTool.exe`, `HwpFormatter.exe`

## 업무 수준
L2 — 서식 수정 초안 생성, 사람이 최종 확인. "회사 표준 적합성" 판단은 L3.

## 운영 방식
서브에이전트로 호출하지 않는다. 지침서(`~/.claude/agents/hwp-format-desk.md`)를
메인 Claude가 직접 따라 실행한다. (서브에이전트는 hwp-mcp 도구를 못 받음)

## 작업 폴더
`업무/` 에 작업물이 쌓인다. 작업 1건 = `업무/YYYY-MM-DD_문서명/` 권장.
