# QBS 작성 매뉴얼 — 소하천정비종합계획(재수립) 류

> 사업수행능력평가서(QBS) 작성 SOP. 표준 출처: 양구군 소하천정비종합계획 QBS (2026.2).
> 발주처마다 평가기준·작성안내서가 오므로, 매 건 그 두 파일을 먼저 파싱해 본 매뉴얼과 대조한다.

## 1. QBS란
입찰 적격심사용 **사업수행능력평가서**. 100점 배점. 발주처가 메일로 보낸
`평가기준 + 작성안내서`에 맞춰 작성 → 일정점수 이상이면 입찰대상자 선정.

## 4. 분야 구성 (소하천정비 = 3분야)
- 수자원개발 70% / 토질·지질 20% / 토목구조 10%
- 인력: 사업책임기술인 1명(대표사 수자원개발) + 분야별책임 3명 + 분야별참여 3명
- 사업책임기술인은 분야별책임·참여와 중복 불가

## 5. 제출 사양
- 2부(원본1·부본1), 원본대조필 날인 / A4 / 백상지 80g, 표지 아트지 200g / 양면좌철(무선철)

## 6. SOP — 작성 절차

| 단계 | 내용 | 누가 / 도구 |
|---|---|---|
| 1 | 받은 메일 파싱 — 평가기준·작성안내서·공고문 → 목차·배점·일정 확인 | Claude (kordoc) |
| 2 | 예시 QBS 중 가장 유사한 건 선택 → 틀 복제 | Claude |
| 3 | 양식 1~7 — 프로젝트 정보 치환(사업명·발주처·일정·회사현황) | Claude (hwp-mcp) |
| 4 | 양식 8~11 — 기술인 자격·경력·실적, 회사 수행실적 입력 | ⚠️ Claude+사람 (실적 선택 판단) |
| 5 | 양식 12~17 — 신용도·기술개발·중복도 (회사 데이터, 갱신 위주) | Claude+사람 |
| 6 | 양식 20 — 기술능력·업무관리능력 평가서 (서술형) | ⚠️ Claude 초안 + 사람 |
| 7 | 일러스트(위치도·공정표) 삽입 → PDF 변환 → 합본 | Claude |
| 8 | 최종 검토·제출 | **사람 (입찰 문서)** |

## 7. 자동화 가능 범위 (솔직)
- ✅ 잘 됨: 양식 1~7 정형 치환, PDF 변환·합본, 목차·배점 대조
- ⚠️ 반자동: 양식 8~11(실적 선택), 양식 20(서술형 초안) — 판단은 사람
- 🔴 사람: 유사실적 최종 선택, 최종 검토·제출 (점수가 수주를 가름)

## 8. 입력 자료 위치 (양구군 예시)
- 받은파일: `...\01_QBS\00_받은파일\[메일]20260226_133530_backup\`
- 예시 QBS: `...\01_QBS\01_예시\` (횡성·세종·원주·영월·홍천·통영)
- 일러스트: `...\01_QBS\02_일러스트\`
- 최종 산출물: `...\01_QBS\(날짜)(사업명)(기술자명).hwp`
