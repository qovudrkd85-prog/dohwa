# 자료다운로드원 — data-download-desk

- **소속**: ③ 토목기술팀 (기상현황조사 파이프라인 **1단계**)
- **하는 일**: 기상청 기상자료개방포털(data.kma.go.kr)에서 크롬으로
  관측자료 CSV 다운로드·정리
- **호출**: "○○ 관측소 △△자료 받아줘" — 또는 "기상현황 분석해줘" 시
  01_weather-desk와 자동 연계(00 다운로드 → 01 분석)
- **업무 수준**: L1 — 다운로드·정리는 혼자 완수
- **에이전트 정의**: `~/.claude/agents/data-download-desk.md`

## 전제 조건
- 디버그 크롬(9222 포트)이 떠 있어야 함 — 바탕화면 `그루웨어-크롬-실행.bat`
- 포털 로그인 필요 — 자료 다운로드 시 data.kma.go.kr 로그인 상태여야 함

## 받기 전 확인 4가지
관측소 / 자료종류(ASOS·AWS) / 단위(분·시·일·월) / 기간 — 전부 입력값.
어떤 조합이든 이 직원 한 명이 처리(자료종류별로 직원 안 나눔).

## 업무 폴더 규칙
다운로드 요청 1건마다 `업무/YYYY-MM-DD_<요청명>/` 폴더 생성:
- 받은 CSV 원본 (가공 안 함 — 분석은 01번 직원 담당)
- `다운로드기록.md` — 관측소·자료종류·단위·기간·받은시각·파일목록
