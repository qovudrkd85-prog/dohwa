# 기상현황조사 분석원 — weather-desk

- **소속**: ③ 토목기술팀 (기상현황조사 파이프라인 2단계)
- **하는 일**: 00_data-download-desk가 받아온 KMA 관측자료 파일을
  기상현황조사 표·차트 Excel 보고서로 추출
- **호출**: 보통 단독 호출 안 함 — "기상현황 분석해줘" 한 마디면
  00 → 01 순서로 자동 진행
- **업무 수준**: L2 — 보고서 생성까지. 설계 적용·해석(L3)은 사람
- **에이전트 정의**: `~/.claude/agents/weather-desk.md`

## 동작 원리
다운로드는 안 한다(00번 직원 담당). 받은 파일만 처리한다.
- 다운로드 CSV 파싱 → rows JSON
- CivilTools `excel_export.py` 재사용 → 표·차트 xlsx
- API 안 씀 → 404 오류 없음

## 업무 폴더 규칙
작업 1건마다 `업무/YYYY-MM-DD_관측소명_기간/` 폴더 생성:
- `rows.json` — 정규화 데이터
- `기상현황조사_관측소명.xlsx` — 보고서
- `작업일지.md` — 입력파일·관측소·기간·핵심수치
