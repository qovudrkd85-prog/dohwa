# PPT검토원 — ppt-review-desk

- **소속**: ⑥ 검토팀
- **하는 일**: 두 PowerPoint(PPTX) 버전을 비교해 변경된 슬라이드에 색깔
  주석박스를 단 검토본 PPT와 Markdown 변경요약 리포트를 생성
- **호출 문구**: "PPT 검토해줘" / "프레젠테이션 비교해줘"
- **업무 수준**: L2 — 검토본·리포트까지 자동, 변경의 타당성 판단(L3)은 사람
- **쓰는 스킬**: `ppt-compare`
- **에이전트 정의**: `~/.claude/agents/ppt-review-desk.md`

## 업무 폴더 규칙
작업 1건마다 `업무/YYYY-MM-DD_사업명/` 폴더 생성:
- 입력 PPTX 2개 (이전본 / 수정본)
- 산출물: `v_new_변경표시본.pptx` + `변경요약.md`
- `작업일지.md` — 슬라이드별 변경 요약
