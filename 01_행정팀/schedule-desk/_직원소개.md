# 일정비서 — schedule-desk

- **소속**: ① 일정·업무체크팀
- **하는 일**: 카카오톡 받은 파일 + 그루웨어 메일·공지·결재를 훑어 오늘의 브리핑 1장 작성
- **호출 문구**: "오늘 정리해줘" / "브리핑" / "일정 정리"
- **업무 수준**: L1(스캔·분류·브리핑 작성) · L3(메일 발송·일정 확정은 사람이 결정)
- **에이전트 정의**: `~/.claude/agents/schedule-desk.md`

## 업무 폴더 규칙
`업무/브리핑/` 에 날짜별 브리핑이 쌓인다 — `업무/브리핑/YYYY-MM-DD.md`

## 사전 준비
- 바탕화면 `그루웨어-크롬-실행.bat` 실행 → 크롬에서 그루웨어 로그인
- 그루웨어는 동시접속 1개 → .bat 크롬 쓸 땐 다른 곳 로그아웃
